Gyulai Márk (NV4AO2) programtervező informatikus jelölt részére.\newline

\noindent\textbf{A szakdolgozat tárgyköre:} Numerikus analízis, Python \newline

\noindent\textbf{A szakdolgozat címe:} Numerikus számítások hatékonyságának vizsgálata Python környezetben\newline

\noindent\textbf{A feladat részletezése:}

\medskip

\emph{A mérnöki gyakorlatban előforduló numerikus számítások jelentős része közvetlenül elérhető a NumPy nevű Python függvénykönyvtárban. A dolgozat célja, hogy bemutassa ezek használatát, elemezze azok megvalósítási módját, pontosságukat és hatékonyságukat. Mivel a függvénykönyvtárban elérhető számítások jellemzően többféle paraméterezéssel rendelkeznek, így részletesen meg kell vizsgálni azok működését, össze kell hasonlítani az alternatív számítási módokat. 
}

\medskip

\emph{A dolgozat tematikája célszerűen illeszkedik a Numerikus módszerek című tárgyban előforduló témákhoz, így a későbbiekben az elkészült leírásokat és példákat fel lehet használni a tananyag kiegészítéseként. A számítások áttekinthetősége érdekében a Python nyelvű forráskódokat érdemes Jupyter munkafüzetekbe rendezni. Az eredmények vizualizálásához a Matplotlib függvénykönyvtár kerül felhasználásra.}
